Data Preprocessing



Data Preprocessing is one of the crucial parts in Machine Learning. When you have the dataset from your business, company, startup, or any institution that you want to use for predicting future outcomes, it is an essential step to prepare the data correctly for machine learning models that you plan to apply to it. To put it simply, you need to have all the ingredients ready-to-use before you start cooking. In this section, I will talk about:

  1. Which libraries you can use in python for Data Preprocessing;
  2. How to read a dataset and create a so-called data frame;
  3. How to handle the problems within your dataset (such as missing data);
  4. How to transform Categorical data into the language that computers can understand;
  5. How to transform independent and dependent variables into the language that computers can understand;
  6. How to split your dataset into training and test sets;
  7. Feature Scaling and usage of it.
